# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2025-12-02

### Added
- Initial release of QuantumDebugger
- Core quantum state simulation engine
- Quantum circuit builder with 15+ standard gates
- Step-through debugger with execution history
- Breakpoint system (gate-based and conditional)
- State inspection and analysis tools
- Circuit profiler with performance metrics
- Visualization tools:
  - State vector plots
  - Probability distributions
  - Bloch sphere (3D)
  - Density matrix visualization
- Example demonstrations:
  - Bell state debugging
  - Grover's algorithm profiling
  - Interactive feature showcase
- Comprehensive documentation and README

### Features
- **Debugging**: Step forward/backward, breakpoints, state inspection
- **Profiling**: Gate analysis, depth calculation, optimization suggestions
- **Visualization**: Matplotlib-based plots for states and circuits
- **Gates**: H, X, Y, Z, S, T, RX, RY, RZ, CNOT, CZ, SWAP, Toffoli
- **Analysis**: Entanglement detection, fidelity, entropy calculations
