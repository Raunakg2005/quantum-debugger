# QML (Quantum Machine Learning) tests

Tests for quantum machine learning algorithms and training.

## Files

- `test_vqe.py` - Variational Quantum Eigensolver tests
- `test_qaoa.py` - QAOA algorithm tests  
- `test_training.py` - Training framework and optimizer tests
- `test_qml_parameterized_gates.py` - Parameterized gate tests
- `test_regression.py` - Regression tests for bugs
- `test_properties.py` - Property-based tests
- `test_edge_cases.py` - Edge case tests

Total: 228 tests covering all QML functionality
